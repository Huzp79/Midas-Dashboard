# 📐 SMC Core Setup

Core setup ตาม Smart Money Concepts ที่ Midas ใช้เป็นหลัก

## Setup Steps
1. **HTF Context** — H4+H1 ต้องตรงทิศกัน
2. **Sweep + CHoCH** — กวาดสภาพคล่อง แล้วโครงสร้างกลับทิศ
3. **FVG + OB** — Fair Value Gap มี Order Block รองรับ
4. **Entry** — Pullback 50% ของ FVG (Consequential Encroachment)
5. **M1 Trigger** — Squeeze Momentum เปลี่ยนสี

## Key Rules
- ไม่เข้าถ้ายังไม่มี Sweep
- ไม่เข้าถ้า Score < 7
- SL ต้องอยู่หลัง Swing High/Low เสมอ
